The Stack Overflow Careers Pattern Library is built upon a 12-column percentage-based grid system. By default columns do not have margins or padding (aka gutters) applied to them. If you wish to have gutters within a grid, you can apply them though. All layout elements have box-sizing: border-box; applied to them so column borders and padding widths are included within a column's width. For more information as to why, please refer to Paul Irish's post, * { Box-sizing: Border-box } FTW.

Since the grid system is percentage-based, all widths are calculated from within their parent containers. Currently the grid system lives within a 1000px div.container element. The grid containers are not responsive yet. (Yes, yes, I know.)
